# libsrun3k

Introduction
------------

`libsrun3k` is a lightweight library for srun3000 [Haut] (http://www.haut.edu.cn/) university of technology [campus network] (http://172.16.154.130/) authentication.

Written in C# and .Net Framework 4.5.2.
